## Prepare

Install the [Parcel](https://parceljs.org/getting_started.html)

```bash
npm install -g parcel-bundler
```

## Development

```bash
# develop index page
parcel app/index.html

# develop config page
parcel app/config.html
```

# Production Build

```bash
npm run build
```
